牙材集中採購管理系統 — 功能總覽
====================================

【登入與權限】
- LINE 一鍵登入 / Email 密碼登入（可記住帳號）
- 角色權限管理（系統管理員 / 總管理者 / 診所管理者）
- 細粒度權限控制（叫貨、收貨、品項管理、價格查看等 14 項權限）

【庫存管理】
- 庫存總覽：各診所即時庫存，安全庫存警示，耗材/非耗材分頁
- 日常變動：進出貨記錄、批次操作、撤銷功能
- 收貨確認：逐項或整批收貨，自動更新庫存

【叫貨 & 訂單】
- 叫貨：品項選擇（格狀/列表）、數量調整、診所選擇
- 貨單管理：訂單狀態追蹤（待確認→已確認→已出貨→已到貨）、逐項狀態更新、自訂價格
- 叫貨總覽：所有訂單總覽與篩選
- 催貨管理：逾期訂單追蹤、催貨通知
- 自動產生叫貨單（一鍵依安全庫存產生）

【多供應商管理】
- 每個品項可設定多個供應商
- 各供應商獨立價格、規格、叫貨單位、換算比例
- 叫貨時選擇供應商

【LINE 通知】
- 低庫存通知（兩階段：安全庫存 & 半數以下）
- 催貨通知
- 每診所指定通知對象
- 通知開關（低庫存/半數/催貨）
- 額外 LINE ID 支援
- 測試通知功能

【報表 & 統計】
- 叫貨統計總覽
- 消耗趨勢分析（圖表）
- 月報表（各診所支出）
- 價格歷史追蹤
- 供應商統計
- 匯出 Excel

【管理設定】
- 品項管理：新增/編輯/刪除品項、圖片、預設安全庫存
- 類別管理：耗材/非耗材分類、排序、拖曳調整
- 廠商管理：供應商資料維護
- 診所管理：多診所管理
- 人員管理：帳號啟用/停用、角色指派
- 角色管理：自訂角色與權限

【客製化】
- 6 色彩主題：深藍/紫/綠/橘/粉/淺色模式
- 背景圖片：各頁面自訂背景、透明度調整
- 泡泡版面編輯：拖曳調整功能泡泡位置，儲存到雲端

【教學文章】
- 教學內容管理（新增/編輯/刪除）
- 圖片燈箱瀏覽
- 作者與時間記錄

【技術架構】
- 單檔 HTML 應用，部署於 GitHub Pages
- 後端：Supabase（PostgreSQL + RLS 資安）
- 圖表：Chart.js
- LINE LIFF SDK 整合
- 響應式設計（桌面/平板/手機）
- localStorage 偏好儲存 + 雲端資料同步
